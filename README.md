# E-commerce UI Automation

A robust UI automation project designed to validate essential flows on [an E-commerce Platform](https://automationexercise.com/), including login, registration, cart operations, and checkout.

## Features & Scope  
- Automates critical user journeys: login, account creation, add-to-cart, checkout  
- Supports both valid, invalid, and edge-case scenarios  
- Uses XML-based test definitions to drive test inputs and expected outcomes  
- Generates HTML reports via FreeMarker templates  
- Maintains test artifacts per user space for isolation  

## Tech Stack  
- Java + Selenium WebDriver  
- TestNG test framework  
- Maven for build & dependency management  
- FreeMarker for HTML report generation  
- XML test definitions for input/expected result data
